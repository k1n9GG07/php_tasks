## Что такое PHP-сервер?

PHP (PHP: Hypertext Preprocessor) — это серверный язык программирования, который выполняется на стороне сервера перед отправкой HTML-страницы браузеру пользователя.

**PHP-сервер** — это комбинация:
- **PHP-интерпретатора** — программы, которая выполняет PHP-код
- **Веб-сервера** (Apache, Nginx или встроенного сервера PHP) — программы, которая обрабатывает HTTP-запросы

Когда браузер запрашивает PHP-файл:
1. Веб-сервер получает запрос
2. PHP-интерпретатор обрабатывает код в файле
3. Результат (обычно HTML) отправляется обратно в браузер

## Как запустить PHP-сервер

### Windows

#### Вариант 1: Встроенный сервер PHP (рекомендуется для обучения)

1. **Установите PHP:**
   - Скачайте PHP с официального сайта: https://windows.php.net/download/
   - Или используйте XAMPP: https://www.apachefriends.org/ (включает PHP, Apache, MySQL)
   - Или установите через Chocolatey: `choco install php`

2. **Проверьте установку:**
   ```powershell
   php -v
   ```

3. **Запустите встроенный сервер:**
   ```powershell
   php -S localhost:8000
   ```

4. **Откройте в браузере:**
   ```
   http://localhost:8000
   ```

#### Вариант 2: XAMPP (полноценный стек)

1. Скачайте и установите XAMPP: https://www.apachefriends.org/
2. Запустите Apache через панель управления XAMPP
3. Поместите файлы в папку `C:\xampp\htdocs\`
4. Откройте `http://localhost/ваш_файл.php`

### Linux

#### Ubuntu/Debian:
```bash
sudo apt update

```

#### Fedora/RHEL:

sudo dnf install php

### macOS

## Введение в синтаксис PHP

### Основные правила

1. **Теги PHP:**
   ```php
   <?php
   // Ваш код здесь
   ?>
   ```

2. **Комментарии:**
   ```php
   // Однострочный комментарий
   # Еще один однострочный комментарий
   
   /*
    * Многострочный
    * комментарий
    */
   ```

3. **Точка с запятой:**
   - Каждая инструкция должна заканчиваться точкой с запятой `;`

4. **Регистр символов:**
   - Имена функций и классов не чувствительны к регистру
   - Имена переменных чувствительны к регистру

5. **Переменные:**
   ```php
   $имя_переменной = значение;
   ```

## Частые ошибки и их решение

### Ошибка: "php не является внутренней или внешней командой"
**Решение:** Добавьте PHP в переменную PATH или используйте полный путь к php.exe

### Ошибка: "Address already in use"
**Решение:** Порт занят. Используйте другой порт: `php -S localhost:8001`

### Ошибка: "Parse error: syntax error"
**Решение:** Проверьте синтаксис файла командой `php -l имя_файла.php`

### Страница не загружается
**Решение:** 
- Убедитесь, что сервер запущен
- Проверьте правильность URL
- Убедитесь, что файл существует и имеет расширение `.php`
