## Задание 1: Простой калькулятор
### Описание
Реализация базового калькулятора для выполнения арифметических операций между двумя числами.

### Инструкции
1. Создать функцию `calculate`, принимающую два числа и строковый оператор.
2. Реализовать логику сложения, вычитания, умножения и деления через конструкцию `switch`.
3. Добавить проверку на деление на ноль с возвратом сообщения об ошибке.
4. Предусмотреть обработку неизвестных операций.

### Технологии
- PHP (базовый синтаксис)
- Управляющие конструкции (`switch`, `if-else`)
- Обработка форм (GET-запросы)

## Задание 2: Работа с данными студентов
### Описание
Разработка набора функций для анализа и манипуляции многомерным массивом данных студентов.

### Инструкции
1. Реализовать функцию `getAverageScore` для вычисления среднего арифметического баллов.
2. Написать функцию `getBestStudent` для поиска записи с максимальным значением балла.
3. Создать фильтр `getStudentsByCourse` для получения списка студентов конкретного курса.
4. Добавить возможность регистрации новых студентов через функцию `addStudent` с передачей массива по ссылке.

### Технологии
- PHP (массивы, циклы `foreach`)
- Работа со ссылками (`&$array`)
- Встроенные функции массивов (`array_column`, `count`)

## Задание 3: Форма обратной связи
### Описание
Создание интерактивной формы с комплексной серверной валидацией пользовательского ввода.

### Инструкции
1. Получить данные из глобального массива `$_POST`.
2. Проверить имя на пустоту и минимальную длину (2 символа).
3. Валидировать формат email с помощью `filter_var`.
4. Реализовать механизм сохранения введенных данных в полях при возникновении ошибок.
5. Применить `htmlspecialchars` для защиты от XSS-атак при выводе данных.

### Технологии
- PHP (обработка POST-запросов)
- Валидация данных (`filter_var`, `mb_strlen`)
- Безопасность (XSS protection)

### Инструкции
1. Реализовать генерацию случайного 5-значного кода капчи.
2. Проверить соответствие введенных учетных данных (`admin`/`12345`) и кода капчи.
3. Организовать доступ к «Личному кабинету» только для авторизованных пользователей.
4. Реализовать механизм выхода из системы с полным уничтожением сессии.

Программная генерация графического файла формата PNG с использованием библиотеки GD.

### Инструкции
1. Создать холст заданного размера (300x150).
2. Выполнить заливку фона и отрисовку геометрических примитивов (прямоугольник, эллипс).
3. Наложить текстовую метку поверх графических элементов.
4. Настроить правильные HTTP-заголовки для отдачи контента как изображения.

### Технологии
- PHP GD Library
- Работа с HTTP-заголовками (`header`)

## Задание 7: Капча-картинка
### Описание
Продвинутая система графической капчи, защищенная от автоматического распознавания шумами и искажениями.

### Инструкции
1. Разделить логику на генератор изображения (`task_07_gen.php`) и страницу формы (`task_07.php`).
2. В генераторе добавить графические шумы (случайные линии).
3. Выводить символы капчи с разным смещением и размером шрифта.
4. Реализовать проверку ввода на стороне сервера с автоматическим обновлением кода после каждой попытки.

### Технологии
- PHP GD (расширенные функции)
- Межстраничное взаимодействие через сессии
- Динамическая генерация ресурсов
